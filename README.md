DESCRIPCION GENERAL DEL SISTEMA:
 Aplicación de escritorio desarrollada en Python con Tkinter para la gestión integral de pedidos de un servicio de delivery. Permite registrar, editar, hacer seguimiento y administrar pedidos de principio a fin, con persistencia de datos en formato JSON.

RAPI DELIVERY simula el flujo completo de una plataforma de delivery: desde la creación de un pedido hasta su entrega, incluyendo cálculo automático de costos de envío, descuentos, tiempos estimados y seguimiento de estado en tiempo real.
